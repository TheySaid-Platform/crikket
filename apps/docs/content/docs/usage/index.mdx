The Crikket Chrome extension captures bug reports from any open tab with
screen recording, console logs, network requests, and error traces attached.

## In This Section

- [Quick Start](/docs/usage/quick-start)
- [Integration](/docs/usage/integration)

## What You Need

- A Crikket workspace
- Chrome or any Chromium-based browser
- The Crikket Chrome extension installed (from the Web Store or unpacked)

## How It Works

1. Install the extension from the Chrome Web Store.
2. Sign in once via the extension popup — it reuses your Crikket session.
3. Press the keyboard shortcut or click the extension to record any tab.
4. Stop the recording; the report uploads to your workspace with logs and
   replay context attached.
