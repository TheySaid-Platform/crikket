## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| `Alt+Shift+R` | Start screen recording |
| `Alt+Shift+S` | Stop the active recording |
| `Alt+Shift+C` | Capture a screenshot |

Rebind these in `chrome://extensions/shortcuts` if they conflict with other
extensions.

## Pointing the Extension At a Self-Hosted Crikket

The Chrome Web Store build targets `https://crikket.io`. To point a build at
your own deployment, build the extension locally with your URLs:

```bash
# apps/extension/.env
VITE_APP_URL=https://app.example.com
VITE_SERVER_URL=https://api.example.com
```

Then build and load the unpacked output — see
[Install Without Web Store](/docs/extension).

## Rolling Out To A Team

- Share the
  [Chrome Web Store install link](https://chromewebstore.google.com/detail/crikket/pjooelapomplepemjoebeanebffdieic)
  in onboarding docs.
- Set a verified domain on your org (in **Settings**) so new teammates
  auto-join when they sign in with a matching email address.
- Reports submitted from the extension land in the same workspace dashboard
  regardless of who captured them.
